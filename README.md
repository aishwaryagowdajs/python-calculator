# 🧮 Python Calculator

This is my first Python project built using Jupyter Notebook.  
It is a simple calculator that performs basic arithmetic operations.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Floor Division

## 📌 Concepts Used
- Variables
- User Input
- If-elif Statements
- Arithmetic Operators

## 💻 How to Run
1. Open the notebook file (calculator.ipynb)
2. Run all cells
3. Enter your choice and numbers when prompted
